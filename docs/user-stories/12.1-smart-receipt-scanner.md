# Story 12.1: Smart Receipt Scanner (OCR + AI)

**Status:** Ready for Implementation
**Epic:** 12 - AI Financial Assistant
**Story Point:** 8 (Complex Integration & Prompt Engineering)

---

## User Story

**As a** busy user,
**I want** to take a photo of my shopping receipt,
**so that** the app can automatically fill in the amount, store name, and category without me having to type manually.

---

## Acceptance Criteria

**AC #1: Receipt Image Capture**

- **Given** layar "Add Transaction",
- **When** user menekan ikon kamera ("Scan Receipt"),
- **Then** buka kamera perangkat menggunakan `image_picker` dan simpan hasilnya di memori sementara.

**AC #2: AI Analysis Overlay**

- **Given** gambar telah diambil,
- **When** data sedang dikirim ke Gemini API,
- **Then** tampilkan _loading animation_ (misal: garis laser yang bergerak naik-turun di atas foto struk) dengan label _"Bob is reading your receipt..."_.

**AC #3: Intelligent Data Mapping (JSON Output)**

- **Given** hasil analisa Gemini,
- **When** data diterima oleh aplikasi,
- **Then** otomatis isi field berikut di form:
  - **Amount:** Angka total belanja (bersihkan dari simbol mata uang).
  - **Notes:** Nama toko/merchandise.
  - **Date:** Tanggal transaksi (konversi ke format `DateTime`).
  - **Category:** Cocokkan dengan daftar kategori user yang sudah ada (fuzzy match).

**AC #4: Validation & Manual Correction**

- **Given** field sudah terisi otomatis,
- **Then** user harus tetap bisa mengedit data tersebut sebelum menekan tombol "Save" (Man-in-the-loop).

**AC #5: Fallback Mechanism**

- **Given** koneksi internet buruk atau AI gagal membaca,
- **Then** tampilkan pesan ramah: _"Aduh, tulisannya agak buram. Bob bantu buka form manual ya!"_

---

## Implementation Details (Winston's Technical Guide)

### 📦 Tools & Packages

- `google_generative_ai`: SDK resmi untuk Gemini.
- `image_picker`: Untuk akses kamera/galeri.
- `flutter_image_compress`: (Opsional) Untuk mengecilkan ukuran gambar sebelum dikirim ke API agar hemat kuota & lebih cepat.

### 🧠 The Gemini Prompt (The Magic Sauce)

Kita akan mengirimkan gambar bersama dengan teks instruksi (Prompt) seperti ini:

> _"Extract data from this receipt. Identify the TOTAL amount, store name, and transaction date. Suggest the best category from this user list: [List_Kategori_User]. Return ONLY a JSON object: {'amount': 55000, 'date': '2026-04-24', 'store': 'Indomaret', 'category': 'Food'}."_

### Tasks / Subtasks

- [ ] **API Setup:** Registrasi API Key di Google AI Studio dan simpan di `.env` (jangan di-hardcode!).
- [ ] **UI Integration:** Tambahkan tombol scan di samping field "Amount" pada form transaksi.
- [ ] **AI Service:** Buat class `GeminiScannerService` untuk menangani komunikasi dengan API.
- [ ] **Parser:** Buat fungsi untuk memetakan respon JSON Gemini ke model `Transaction` Anda.

---

## Technical Summary

Gunakan model **Gemini 1.5 Flash** (bukan Pro) untuk fitur ini. Flash jauh lebih murah/gratis (dalam kuota tertentu) dan sangat cepat untuk tugas ekstraksi teks dari gambar. Pastikan Anda melakukan kompresi gambar ke kualitas 50-70% agar payload tidak terlalu besar.
