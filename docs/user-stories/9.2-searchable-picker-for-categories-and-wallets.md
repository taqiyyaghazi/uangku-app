# Story 9.2: Searchable Picker for Categories & Wallets

**Status:** Ready for Implementation
**Epic:** 9 - Power User Experience (UX Refactor)
**Story Point:** 5 (Component Refactor & Search Logic)

---

## User Story

**As a** power user with many categories and wallets,
**I want** to pick a category or wallet using a searchable list in a bottom sheet,
**so that** I can find exactly what I need without scrolling through long rows of chips.

---

## Acceptance Criteria

**AC #1: Bottom Sheet Trigger**

* **Given** layar input atau edit transaksi,
* **When** user menekan area "Category" atau "Wallet",
* **Then** munculkan `ModalBottomSheet` yang menutupi kira-kira 50-70% layar.

**AC #2: Real-time Search Bar**

* **Given** Bottom Sheet terbuka,
* **When** user mengetik di *Search Bar* yang ada di bagian atas sheet,
* **Then** filter daftar kategori/wallet secara instan berdasarkan nama yang diketik.

**AC #3: Visual List Organization**

* **Given** daftar pilihan di dalam sheet,
* **Then** tampilkan setiap pilihan dalam baris vertikal (List Tile) yang mencakup:
* Icon Kategori/Wallet.
* Nama (Highlight teks yang cocok dengan pencarian).
* Saldo saat ini (Khusus untuk Wallet Picker).



**AC #4: Rapid Selection & Auto-Close**

* **Given** user menemukan kategori yang dicari,
* **When** baris tersebut di-tap,
* **Then** simpan pilihan ke form utama dan tutup Bottom Sheet secara otomatis.

---

## Implementation Details (Winston's Technical Guide)

### 🛠️ UI Refactor

* **Old:** `SingleChildScrollView` + `Wrap` / `ActionChip`.
* **New:** `TextFormField` (Read-only) sebagai pemicu ➡️ `showModalBottomSheet` ➡️ `StatefulBuilder` (untuk internal search state).

### 📦 Logic & Search

Gunakan filter sederhana pada List:

```dart
final filteredItems = allItems.where((item) => 
  item.name.toLowerCase().contains(_searchQuery.toLowerCase())
).toList();

```

### Tasks / Subtasks

* [ ] **Component Creation:** Buat widget `SearchablePickerSheet` yang reusable untuk Kategori maupun Wallet.
* [ ] **Integration:** Ganti UI chip lama di `AddTransactionScreen` dengan widget pemilih baru ini.
* [ ] **Keyboard Management:** Pastikan *Search Bar* otomatis fokus (`autofocus: true`) saat sheet dibuka agar user bisa langsung mengetik.
