# Story 10.1: Wallet Management Hub & Quick Search

**Status:** Ready for Implementation
**Epic:** 10 - Advanced Discovery & Deep Filtering
**Story Point:** 3 (UI Layout & Navigation)

---

## User Story

**As a** power user with many wallets,
**I want** to have a dedicated screen to view, search, and add my wallets,
**so that** I don't have to scroll horizontally to the very end just to find a specific wallet or add a new one.

---

## Acceptance Criteria

**AC #1: Dashboard "See All" Entry Point**
- **Given** Section "My Wallets" di Dashboard,
- **When** user melihat judul section tersebut,
- **Then** tampilkan tombol "See All" atau icon "Settings/Manage" (⚙️) di samping kanan judul.

**AC #2: Wallet List Screen (Vertical)**
- **Given** user menekan tombol "See All",
- **When** layar baru terbuka,
- **Then** tampilkan seluruh wallet dalam format **List Vertikal** yang rapi (bukan carousel/grid) agar mudah di-scan mata.

**AC #3: Search Bar Integration**
- **Given** layar Wallet List,
- **When** user mengetik di kolom pencarian di bagian atas,
- **Then** filter daftar wallet secara real-time berdasarkan nama wallet.

**AC #4: Primary Add Wallet Button**
- **Given** layar Wallet List,
- **When** user ingin menambah wallet baru,
- **Then** sediakan Floating Action Button (FAB) atau tombol "Add Wallet" yang menetap (Static) di layar tersebut, sehingga tidak tersembunyi di ujung scroll manapun.

---

## Implementation Details (Winston's Technical Guide)

### 🛠️ UI Components
- **Dashboard Header:** Gunakan `Row` dengan `MainAxisAlignment.spaceBetween` untuk menaruh Judul dan tombol "See All".
- **WalletListScreen:** - Gunakan `Scaffold` dengan `AppBar` yang berisi `SearchField`.
    - Gunakan `ListView.separated` untuk menampilkan wallet.
    - Setiap item menggunakan `ListTile` kustom yang menampilkan: Icon Wallet, Nama, dan Saldo (Format mata uang).

### Tasks / Subtasks
- [ ] **Navigation:** Tambahkan route baru `WalletListScreen`.
- [ ] **Search Logic:** Implementasikan `TextEditingController` dan filter `List<Wallet>` secara lokal (Client-side filtering sudah cukup cepat untuk < 100 wallet).
- [ ] **Refactor Add Button:** Pindahkan atau duplikasi logika `AddWallet` dari carousel dashboard ke FAB di layar baru ini.



---

## Technical Summary
Pastikan saat user kembali dari `AddWalletScreen` ke `WalletListScreen`, daftar wallet otomatis terperbarui. Karena Anda sudah menggunakan **Drift (Stream)** atau **Firestore (Snapshot)**, hal ini seharusnya sudah berjalan otomatis selama Anda memantau *stream* data yang sama.
