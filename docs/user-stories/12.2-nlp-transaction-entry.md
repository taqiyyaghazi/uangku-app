# Story 12.2: NLP Transaction Entry (Chat-to-Record)

**Status:** Ready for Implementation
**Epic:** 12 - AI Financial Assistant
**Story Point:** 5 (Prompt Engineering & Context Mapping)

---

## User Story

**As a** user who wants to track expenses on the go,
**I want** to type or speak a natural sentence (e.g., "Tadi bayar kopi 25rb pakai Gopay"),
**so that** the app can automatically parse the information and save the transaction without me filling out a complex form.

---

## Acceptance Criteria

**AC #1: Magic Input Field**

- **Given** Dashboard Utama,
- **When** user menekan area input "Quick Entry" (mirip chat bar),
- **Then** izinkan user mengetik kalimat dalam bahasa Indonesia atau Inggris secara bebas.

**AC #2: Intent & Entity Extraction**

- **Given** kalimat input dari user,
- **When** dikirim ke Gemini,
- **Then** AI harus mengembalikan data terstruktur:
  - **Amount:** Ekstrak angka (misal: "25rb" jadi `25000`).
  - **Type:** Tentukan apakah ini _Expense_, _Income_, atau _Transfer_.
  - **Wallet:** Cocokkan dengan nama wallet user (misal: "Gopay", "BCA", "Cash").
  - **Category:** Prediksi kategori yang paling sesuai dari daftar kategori milik user.
  - **Note:** Ambil deskripsi transaksi (misal: "bayar kopi").

**AC #3: Smart Confirmation Dialog**

- **Given** data hasil ekstraksi AI,
- **When** data siap,
- **Then** tampilkan _Summary Card_ kecil untuk dikonfirmasi user: _"Bob mencatat: Pengeluaran 25.000 untuk Kopi dari Gopay. Benar?"_
- **Action:** Klik "Simpan" untuk masuk ke database, atau "Edit" untuk membuka form manual.

**AC #4: Contextual Defaulting**

- **Given** kalimat yang tidak lengkap (misal: "Makan bakso 15000"),
- **When** wallet tidak disebutkan,
- **Then** gunakan _Default Wallet_ milik user atau minta klarifikasi dengan sopan.

---

## Implementation Details (Winston's Technical Guide)

### 🧠 The "Context-Heavy" Prompt

Agar Gemini akurat, kita harus mengirimkan "Dunia" user kepadanya:

> _"Daftar Wallet User: [BCA, Gopay, Cash]. Daftar Kategori: [Makanan, Transportasi, Hiburan]. Kalimat User: 'Tadi isi bensin 100rb pake BCA'. Analisa kalimat tersebut dan kembalikan ONLY JSON: {'amount': 100000, 'type': 'expense', 'wallet': 'BCA', 'category': 'Transportasi', 'note': 'isi bensin'}"_

### 📋 UI Component

- Gunakan `TextField` yang didesain menyerupai chat bubble di bagian bawah dashboard atau di dalam FAB yang bisa di-expand.
- Gunakan `TypeWriter` animation saat AI sedang "berpikir" untuk memberikan kesan interaktif.

---

## Technical Summary

Gunakan model **Gemini 1.5 Flash**. Untuk Story 12.2 ini, latensi (kecepatan respon) sangat krusial agar user tidak merasa menunggu lama hanya untuk mencatat satu baris. Pastikan Anda menangani variasi bahasa gaul (misal: "ceban", "goceng", "seratus rebu") karena Gemini sangat jago menangani slang Indonesia.

## Context References

**UX-Spec Update:**
