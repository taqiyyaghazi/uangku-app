# Story 11.2: Global Privacy Mode (Hide Balances)

**Status:** Completed
**Epic:** 11 - Data Privacy & Session Security
**Story Point:** 3 (State Management & UI Implementation)

---

## User Story

**As a** privacy-conscious user,
**I want** to be able to hide all nominal values (balances, transactions, and investments) with a single toggle,
**so that** I can open the app in public places without worrying about prying eyes.

---

## Acceptance Criteria

**AC #1: Global Toggle Switch**

- **Given** di Header Dashboard atau di halaman Settings,
- **When** user menekan ikon "Mata" (Eye Icon),
- **Then** ganti status privasi antara **Visible** dan **Hidden**.

**AC #2: Universal Masking**

- **Given** status privasi adalah **Hidden**,
- **Then** semua nilai nominal di seluruh aplikasi (Dashboard, Wallet Hub, History, Investment Snapshot, Budget) harus berubah menjadi teks sensor (misal: `Rp ****` atau `••••`).

**AC #3: Persistence Across Sessions**

- **Given** user mengaktifkan fitur Hide Nominal,
- **When** aplikasi ditutup dan dibuka kembali,
- **Then** status privasi harus tetap aktif (Simpan status di `SharedPreferences`).

**AC #4: Visual Feedback**

- **Given** ikon pemicu (Eye Icon),
- **When** mode Hidden aktif, gunakan ikon mata tertutup (`Icons.visibility_off`).
- **When** mode Visible aktif, gunakan ikon mata terbuka (`Icons.visibility`).

---

## Implementation Details (Winston's Technical Guide)

### 🛠️ State Management

Gunakan **Provider**, **Bloc**, atau **Signals** untuk memantau status `isBalanceHidden`.

```dart
class PrivacyProvider extends ChangeNotifier {
  bool _isHidden = false;
  bool get isHidden => _isHidden;

  void togglePrivacy() {
    _isHidden = !_isHidden;
    // Simpan ke SharedPreferences
    prefs.setBool('is_hidden', _isHidden);
    notifyListeners();
  }
}
```

### 📋 UI Extension (The "Masking" Widget)

Buat widget pembungkus agar koding tidak berulang:

```dart
String formatAmount(double amount, bool isHidden) {
  if (isHidden) return "Rp ••••••";
  return CurrencyFormatter.format(amount); // Fungsi format rupiah Anda
}
```

### Tasks / Subtasks

Tantangan utamanya adalah **Konsistensi**. Pastikan tidak ada "kebocoran" nilai, misalnya di dalam _Charts_ (Story 5.1) atau di _Bottom Sheet_ (Story 9.2). Jika sedang mode Hidden, angka di sumbu grafik atau total di picker juga harus disensor atau disederhanakan.
