# Research Log: Smart Receipt Scanner (Epic 12.1)

## Overview
This feature allows users to capture a photo of a shopping receipt and uses the Gemini API (specifically `gemini-1.5-flash`) to perform OCR and extract structured JSON data to auto-fill a transaction form.

## Dependencies Needed
1. `image_picker`: For accessing the device camera and picking images.
2. `google_generative_ai`: Official SDK for Gemini API integration.
3. `flutter_image_compress` (optional but recommended): To reduce image size before sending it to Gemini to save bandwidth and reduce latency.

## Implementation Details

### Image Capture
Use `ImagePicker` to allow the user to take a photo:
```dart
final ImagePicker picker = ImagePicker();
final XFile? image = await picker.pickImage(source: ImageSource.camera);
if (image != null) {
  final bytes = await image.readAsBytes();
  // Pass bytes to Gemini
}
```

### Gemini Setup
Use `google_generative_ai` package to send the prompt and image.
We will use the `gemini-1.5-flash` model for fast multimodal responses.

```dart
final model = GenerativeModel(
  model: 'gemini-1.5-flash',
  apiKey: apiKey,
  generationConfig: GenerationConfig(responseMimeType: 'application/json'),
);
final content = [
  Content.multi([
    TextPart('Extract data from this receipt. Identify the TOTAL amount, store name, and transaction date. Suggest the best category from this user list: [List_Kategori_User]. Return ONLY a JSON object: {"amount": 55000, "date": "2026-04-24", "store": "Indomaret", "category": "Food"}.'),
    DataPart('image/jpeg', bytes),
  ])
];
final response = await model.generateContent(content);
```
